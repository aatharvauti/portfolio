Installing Java on Arch BTW

Java comprises JRE (Java Runtime Environment) and JDK (Java Development Toolkit).

JRE is a set of software applications that help in the deployment of Java applications. JDK is a development environment necessary for the building and compilation of Java applications.

To install java on Arch Linux, simply paste the following command in your terminal:

sudo pacman -Sy jdk-openjdk jre-openjdk
